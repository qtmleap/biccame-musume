# イベント一覧ページ UI リファクタリング計画

対象: `src/app/routes/events/index.tsx` および `src/components/events/*`

日付: 2026-04-11

## 背景

`/ui-refactor events` によるオーディット結果。モバイルのフィルター UX 改善（リセットボタン統合、タップターゲット拡大、Sheet 高さ修正、フィルター有効インジケーター）は本計画から切り出して先行実施・コミット済み。本ドキュメントには **残りの改善項目**を記録する。

関連コミット:
- dark: クラス除去（無効なダークモード指定の一掃）
- mobile filter drawer UX 改善

### 推奨変更（優先順）

1. **[中] カードスタイル統一（#7, #11）** — 終了間近色をテーマ準拠に、hover に軽い elevation
2. **[中] 空状態の改善（#9）** — CTA「フィルターをクリア」を空状態ボックスに追加
3. **[中] カテゴリフィルタ Chip 化（#10）** — 選択状態を視覚的に強化
4. **[中] フィルタコンポーネント重複解消（#5）** — map 化、トークン化
5. **[中] 件数表示の圧縮（#8）**
6. **[低] コンテナ幅調整（#12）**
7. **[低] 管理ボタンのモバイルラベル隠し（#13）**

### 除外項目（対応不要と判断）

- **#1, #2（テーマトークン統一・ダークモード）** — 本プロジェクトはダークモード未使用方針のため対応不要。代わりに無効な `dark:` 指定を一掃するコミットを先行実施済み。
- **#3, #4, #6（モバイルフィルター UX 改善）** — 先行コミット済み。
  - モバイルリセットボタンの独立行を削除
  - Sheet フッターに全幅クリアボタン統合
  - フィルター有効時の赤ドットインジケーター
  - Sheet の高さ計算修正（`h-auto max-h-[85vh] flex flex-col`）
  - タップターゲット 44px 化

## 制約・ルール

- 純黒（`#000`）・純白（`#fff`）は使わない（feedback_no_pure_black_white.md 参照）
- shadcn/ui コンポーネント（`src/components/ui/*`）は直接編集しない — className で上書き
- アニメーションは既存モーションパターンに合わせる
- 変更後は Playwright でモバイル（375px）とデスクトップ（≥1280px）のスクリーンショット確認

**M**（ファイル 5–6、機能変更なし、既存ロジック流用）
